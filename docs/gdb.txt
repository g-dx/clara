View memory: x/nfu <addr>
 n = no of bytes
 f = `s' (null-terminated string) or `i' (machine instruction). The default is `x' (hexidecimal)
 u = unit size b (Bytes), h (two bytes), w (four bytes), g (eight bytes).

Ex: x/s <addr> = print as string
